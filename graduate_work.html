GIS 5571: Intermediate Skills in Spatial Data Science

Course Description: This course focuses on teaching intermediate skills in spatial data science using the Esri ecosystem of geospatial technologies. Students gain skills to build data integration pipelines and perform reproducible deterministic spatial analyses. The course aims to equip students with the ability to build geospatial workflows and data systems to address intermediate-level spatial analysis problems.

Learning Objectives:

  • Work with ArcPro, Python, Jupyter Notebooks, and supporting open source tools.
  • Create and edit various data types including vector, raster, tabular, geojson, spacetime cubes, TINs, terrain, and networks.
  • Perform intermediate deterministic spatial analysis on multiple data types.
  • Optimize routes and solve spatial allocation problems.
  • Describe the spatial computing algorithms that underlie Esri toolsets.
  • Create professional-level spatial data science documentation and workflows.

Selected Labs - Lab 2

Part 1: This section involves downloading and converting .las data to create DEM and TIN files, followed by exporting these files as PDFs. The process includes working with various geospatial data formats and performing spatial data transformations.

Part 2: This part focuses on working with landcover and elevation data. It includes downloading and reclassifying landcover data, calculating slopes, and generating cost surfaces based on weighted combinations of landcover and slope data. The end goal is to create multiple cost surfaces for spatial analysis.
