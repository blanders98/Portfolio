MGIS Class Work

Overview: Selected classes, labs, and projects from my time in the MGIS program at the University of Minnesota.

GIS 5571: Intermediate Spatial Data Science Applied with ArcGIS

Course Description: This course focuses on teaching intermediate skills in spatial data science using the Esri ecosystem of geospatial technologies. Students gain skills to build data integration pipelines and perform reproducible deterministic spatial analyses. The course aims to equip students with the ability to build geospatial workflows and data systems to address intermediate-level spatial analysis problems.

Selected Labs - Lab 2

Part 1: This lab involves downloading and converting .las data to create DEM and TIN files, followed by exporting these files as PDFs. The process includes working with various geospatial data formats and performing spatial data transformations.

Part 2: This part of the lab focuses on working with landcover and elevation data. It includes downloading and reclassifying landcover data, calculating slopes, and generating cost surfaces based on weighted combinations of landcover and slope data. The end goal is to create multiple cost surfaces for spatial analysis.

GIS 5574: Web GIS and Services

Course Description: This course empowers students to plan, design, and publish web-based GIS and mapping solutions. It provides hands-on experience with various Web GIS technologies, from commercial to open source, and covers topics such as cloud service models, online web GIS platforms, web GIS services, web mapping libraries, mapping APIs, and web mapping applications.
