About this Portfolio

This portfolio was created to showcase my work in graduate studies, research, and professional experience. It is developed using HTML, CSS, and JavaScript to ensure a seamless and interactive experience for the reader.

HTML

The structure of this site is built using HTML. Each page consists of elements such as headers, sections, and footers to organize the content in a logical, easy to read way. Navigation links are included to allow easy access to different sections of the portfolio.

CSS

CSS is used to style the HTML content. The styling includes a consistent layout with a white box background, headers, navigation menus, and footers. Flexbox is used to manage the layout.

JavaScript

JavaScript adds another layer to the portfolio by employing the use of OpenStreetMap as a background. The map is rendered using the Leaflet.js library.
